 Crest - Smart Project Management System

Crest is a full-stack project management system built using the MERN stack. It allows teams to create projects, assign tasks, track progress, and collaborate in real time.

Features
- Project and task management
- Kanban board
- Real-time updates
- Time tracking
- Risk detection
- Role-based access (Manager and User)

Tech Stack
- React
- Node.js
- Express
- MongoDB
- Socket.io

Roles

Manager
- Create projects
- Assign tasks
- Monitor progress

User
- View assigned tasks
- Update status
- Add comments

Installation

Backend
cd backend
npm install
npm run dev

Frontend
cd frontend
npm install
npm run dev

Environment Variables

Backend
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret

Frontend
VITE_API_URL=your_backend_url
